calculadora.py: Contém a lógica da calculadora, incluindo a validação de notas e o cálculo da média ponderada.
test_calculadora.py: Conjunto de testes para garantir que o código funcione.
requirements.txt: Lista as dependências necessárias para o projeto, especificamente o framework pytest e a ferramenta de cobertura pytest-cov.
.gitignore: Indica ao Git quais pastas (como ambientes virtuais e caches de teste) devem ser ignoradas no controle de versão.
